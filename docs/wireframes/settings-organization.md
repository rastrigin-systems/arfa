# Organization Settings Page - Wireframe

## Purpose

Allow organization administrators to view and edit organization details, and view subscription information.

## User Story

**As an organization admin, I want to:**
- Update organization name and description
- View subscription details (plan, usage, billing period)
- See organization metadata (created/updated timestamps)
- Save or cancel changes with clear feedback

## Component Specifications

### Page Header

**Breadcrumb:** Settings > Organization
**Title:** Organization Settings
**Subtitle:** Manage your organization details and subscription

**Styling:**
- Title: `text-3xl font-bold`
- Subtitle: `text-muted-foreground text-base`
- Spacing: `space-y-1 mb-8`

### Organization Information Card

**Type:** Form section with Card component

**Fields:**

1. **Organization Name** (Required)
   - Type: Text input
   - Max length: 255 characters
   - Validation: Required, min 3 chars
   - Helper: "Your organization's display name"

2. **Description** (Optional)
   - Type: Textarea
   - Rows: 4
   - Max length: 1000 characters
   - Helper: "Brief description of your organization (optional)"

3. **Metadata** (Read-only)
   - Created: Timestamp (formatted: "January 15, 2024, 10:30 AM")
   - Last Updated: Timestamp (formatted: "December 10, 2024, 3:45 PM")
   - Styling: `text-sm text-muted-foreground`

**Layout:**
- Card with padding: `p-6`
- Fields spacing: `space-y-4`
- Separator before metadata: `border-t pt-4 mt-4`

### Subscription Card

**Type:** Read-only Card component

**Content:**

1. **Plan Badge**
   - Display: Badge component
   - Variants:
     - Starter: `variant="secondary"`
     - Professional: `variant="default"`
     - Enterprise: `variant="premium"`

2. **Budget Information**
   - Monthly Budget: Currency format ($500.00)
   - Current Spending: Currency format ($327.45)
   - Typography: `text-base font-medium`

3. **Progress Bar**
   - Width: 100%
   - Height: 8px
   - Filled: `bg-primary` (percentage based on spending/budget)
   - Empty: `bg-muted`
   - Percentage label: `text-sm font-medium` (e.g., "65%")

4. **Billing Period**
   - Format: "December 1, 2024 - December 31, 2024"
   - Typography: `text-sm`

5. **Next Renewal**
   - Format: "January 1, 2025"
   - Typography: `text-sm`

6. **Footer Note**
   - Text: "Need to change your plan? Contact sales@ubik.ai"
   - Typography: `text-sm text-muted-foreground`
   - Link: `text-primary hover:underline` on email

**Layout:**
- Card with padding: `p-6`
- Nested card for plan details: `bg-muted p-4 rounded-lg`
- Spacing: `space-y-3`

### Action Buttons

**Cancel Button:**
- Variant: `variant="outline"`
- Enabled: Only when form is dirty
- Action: Reset form to original values
- Shortcut: Esc key

**Save Changes Button:**
- Variant: `variant="default"`
- Enabled: Only when form is dirty AND valid
- Action: Submit form, call PATCH /organizations/current
- Loading state: Spinner + "Saving..."

**Layout:**
- Container: `flex gap-3 mt-8`
- Alignment: Left-aligned (not right)

// app/(dashboard)/settings/organization/page.tsx
export default async function OrganizationSettingsPage() {
  const token = await getServerToken()

  // Parallel fetch
  const [orgRes, subRes] = await Promise.all([
    apiClient.GET('/organizations/current', {
      headers: { Authorization: `Bearer ${token}` }
    }),
    apiClient.GET('/subscriptions/current', {
      headers: { Authorization: `Bearer ${token}` }
    }),
  ])

  return (
    <div className="space-y-8">
      <PageHeader
        title="Organization Settings"
        subtitle="Manage your organization details and subscription"
      />

      <OrganizationInfoForm
        initialData={orgRes.data.organization}

      <SubscriptionCard

  )

### Form Component (Client Component)

```typescript
// components/settings/OrganizationInfoForm.tsx
'use client'

import { useForm } from 'react-hook-form'
import { zodResolver } from '@hookform/resolvers/zod'
import { z } from 'zod'

const schema = z.object({
  name: z.string().min(3).max(255),
  description: z.string().max(1000).optional(),
})

export function OrganizationInfoForm({ initialData }) {
  const form = useForm({
    resolver: zodResolver(schema),
    defaultValues: {
      name: initialData.name,
      description: initialData.settings.description || '',
    },
  })

  const isDirty = form.formState.isDirty
  const isValid = form.formState.isValid

  // Warn on navigation if dirty
  useEffect(() => {
    const handleBeforeUnload = (e: BeforeUnloadEvent) => {
      if (isDirty) {
        e.preventDefault()
        e.returnValue = ''
      }
    }
    window.addEventListener('beforeunload', handleBeforeUnload)
    return () => window.removeEventListener('beforeunload', handleBeforeUnload)
  }, [isDirty])

  const onSubmit = async (data) => {
    // API call
  }

  return (
    <Card>
      <CardHeader>
        <CardTitle>Organization Information</CardTitle>
      </CardHeader>
      <CardContent>
        <Form {...form}>
          <form onSubmit={form.handleSubmit(onSubmit)} className="space-y-4">
            {/* Form fields */}

            <div className="flex gap-3 mt-8">
              <Button
                type="button"
                variant="outline"
                disabled={!isDirty}
                onClick={() => form.reset()}
              >
                Cancel
              </Button>
              <Button
                type="submit"
                disabled={!isDirty || !isValid}
              >
                Save Changes
              </Button>
            </div>
          </form>
        </Form>
      </CardContent>
    </Card>
  )
}
```
