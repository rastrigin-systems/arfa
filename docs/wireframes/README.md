# Wireframes

This directory contains wireframes for all UI pages.

## Naming Convention
- Use descriptive names: `page-name.png` or `page-name.jpg`
- For page updates, include version: `page-name-v2.png`

## Required for:
- ✅ New page creation
- ✅ Existing page modifications

## Tools
Use any wireframing tool:
- Figma, Sketch, Adobe XD (design tools)
- Balsamiq, Wireframe.cc (wireframe tools)
- Hand-drawn sketches (photographed/scanned)
- Screenshots with annotations

## Workflow
1. Create wireframe before implementing UI
2. Get approval/review if needed
3. Implement UI matching wireframe
4. Update wireframe if design changes during implementation
5. Include wireframe in PR/commit
