## Accessibility (Mobile)

### Touch Accessibility
- Minimum 44x44px touch targets
- Adequate spacing between tappable elements (8px minimum)
- No overlapping touch zones

- Actions button: `aria-label="Actions for {agent}"`

- Disable card animations
- Instant transitions instead of slides

### Dark Mode (Bonus)

## Performance Optimizations

### Lazy Loading
- Load 10 cards initially
- Infinite scroll or "Load More" button
- Placeholder cards during fetch

### Image/Icon Optimization
- Use SVG for icons (crisp at any size)
- Lazy load avatar images
- Compress any raster images

### Network Awareness
- Cache configs locally (IndexedDB)
- Show cached data immediately
- Refresh in background
- Offline indicator if no connection

### Animation Performance
- Use CSS transforms (GPU-accelerated)
- Avoid layout thrashing
- `will-change` for animated properties
- 60fps target

## Edge Cases (Mobile)

### Small Screens (320px)
- Reduce padding: 12px instead of 16px
- Smaller font sizes: -1px
- Single column always
- Truncate long text more aggressively

### Large Phones (428px)
- Increase card width slightly
- More breathing room
- Consider 2-column layout in landscape

### Keyboard Open
- Shrink modal height to fit
- Scroll to focused input
- Dismiss keyboard on scroll

### Orientation Change
- Save scroll position
- Recalculate card layout
- Adjust modal size

### Slow Network
- Show loading state immediately
- Timeout after 30 seconds
- Allow retry without full page reload
- Cache aggressively
