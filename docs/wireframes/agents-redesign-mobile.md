## Accessibility (Mobile)

### Touch Feedback
- **Ripple effect** on button press (Material Design style)
- **Haptic feedback** on toggle switch (iOS/Android)
- **Visual press state** - Button darkens 10% on active

### Screen Reader (VoiceOver/TalkBack)
```
"Claude Code. IDE Agent. AI-powered CLI coding assistant. Enabled. Double-tap to configure."
"Cursor. IDE Agent. AI pair programming in VS Code. Disabled. Double-tap to enable."
```

### Zoom Support
- All text scales with system font size
- Layout reflows for 200% zoom
- No horizontal scrolling required

### Motion
- Respects `prefers-reduced-motion`
- Disables slide animations if user has motion sensitivity
- Instant state changes instead of transitions

---

## Performance (Mobile)

1. **Lazy load images** - Agent logos only load when in viewport
2. **Infinite scroll** - Load 10 cards at a time
3. **Optimistic updates** - Toggle immediately, rollback on error
4. **Offline support** - Show cached agents, indicate stale data
5. **Low bandwidth** - Progressive image loading, reduce animations
