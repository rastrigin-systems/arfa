## Responsive Breakpoints

- **Desktop (1024px+)**: 3-column grid, full sidebar
- **Tablet (768px-1023px)**: 2-column grid, collapsible sidebar
- **Mobile (320px-767px)**: 1-column list, hamburger menu

See [agents-redesign-mobile.md](./agents-redesign-mobile.md) for mobile wireframes.

## Accessibility

### Keyboard Navigation
- `Tab`: Navigate between cards
- `Enter/Space`: Activate focused button/toggle
- `/`: Focus search input
- `Esc`: Close dialogs

### Focus Management
- Visible focus ring (2px blue-600, 4px offset)
- Focus trap in confirmation dialog
- Return focus to trigger after dialog close

### Screen Reader Announcements
- "Claude Code enabled for organization" (on enable)
- "Showing 6 of 6 agents" (on filter change)
- "Loading agents" (during fetch)

## Performance Considerations

1. **Lazy load agent descriptions** - Only load full text on expand
2. **Optimistic UI updates** - Toggle immediately, rollback on error
3. **Debounced search** - 300ms delay on search input
4. **Virtual scrolling** - If catalog exceeds 50 agents
